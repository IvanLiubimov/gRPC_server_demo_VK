VK_demo

Проект реализует gRPC сервис для хранения ключ-значение на Java с использованием Tarantool. 
Для запуска базы используйте файл src/main/resources/kv_init.lua, который создаёт пространства KV и KV_2, а также пользователя test с паролем test. 
После запуска Tarantool сервис можно запускать через java -jar target/kv-grpc-service.jar и использовать методы put, get, delete и range.
Проект реализован в объектно-ориентированном стиле, покрыт интеграционными тестами.
